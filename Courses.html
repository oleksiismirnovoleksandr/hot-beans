Beginner

With the beginner's course, you will learn how to create your very own website with the following skills:

HTMl
CSS
JavaScript

With those skills combined, you will create your own portofolio website as a project as you learn each building block of website development.
As you learn more skills along the way, you may even want to recreate your portofolio website or add your skills to it.

Requirement

There is no need to know coding at all, we will teach you what you need along with any questions.
You are to be aged 16 or over
Good understanding of the English language (Higher or equal to GSCE 4 English or equivelant certification)
