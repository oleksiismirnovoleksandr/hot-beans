Frontend Developer

A frontend developer is responsible for what the users see and interact with on the website which is the User Interface (UI). Polishing and determines what each part of the website or application should look to the user. This also includes making the website intuitive to navigate.

Skills

HTML (Hypertext Markup Language) is responsible for the basic building block needed to develop websites; much like how it is responsible as to how you are seeing this section within this section as of now.
CSS (Cascading Style Sheets) is used to create the layout, color and overal style of the website. Much how you see this card highlighted as you read it.
JavaScript is responsbile for the functionality of the website. To create certain functions for the mobile version of this website; create a night and light color theme which you will see with some websites. There are possibilities restricted to your imagination with what Javascript can do.

Backend Developer

Backend development is responsbile for working on the server-side software. Essentially everything you do not see on the website. This includes credit Card Transactions; Databases; Application Programming Interface (API); Servers and much more. Backend development is the back bone of all websites.
